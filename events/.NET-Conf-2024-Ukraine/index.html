.NET Conf 2024 Ukraine

Дніпро, просп. Д.Яворницького, 19, НТУ «Дніпровська політехніка», 1 корпус, ауд.94

12/17/2024 4:00:00 PM +02:00

Price: 0 UAH

Запрошуємо вас на сесію, присвячену технологіям Microsoft .NET та іноваційному продукту World Explorer на базі .NET 9, що пропонує реальні рішення для мандрівників та дослідників. Під час заходу будуть детально розглянуті ключові аспекти архітектури та новітні технології й інструменти.

Основні теми сесії:

Архітектура проєкту:

  • Розглянемо модульний моноліт, чисту архітектуру та архітектуру вертикальних зрізів. Обговоримо їх переваги та виклики, використовуючи реальні приклади з розробки.

SAGA та механізми Inbox/Outbox:

  • Поглибимося в патерн SAGA для управління транзакціями в мікросервісах та розглянемо, як Inbox/Outbox забезпечують надійність у складних системах.

Огляд новинок .NET 9:

  • Обговоримо нові функції, які підвищують продуктивність, включаючи підтримку Ahead-of-Time (AOT) компіляції.

Інтеграція EF Core з NetTopology:

  • Дослідимо, як Entity Framework Core може працювати з геопросторовими даними.

Кросплатформні фреймворки для UI:

  • Проведемо порівняння .NET MAUI, UNO та Avalonia, щоб допомогти обрати найбільш підходящий інструмент для проєкту.

Представлення .NET ASPIRE:

  • Оглянемо новий фреймворк для створення продуктивних і масштабованих додатків та його ключові функції.

Штучний інтелект за допомогою ML.NET:

  • Продемонструємо можливості ML.NET для створення AI-рішень у .NET, порівняно з традиційними підходами.

Процес деплою в маркетплейси:

  • Розглянемо етапи розгортання .NET-додатків у Google Play та Microsoft Store, з акцентом на виклики та особливості.

GraphQL в .NET:

  • Обговоримо GraphQL як потужний інструмент для створення API, який забезпечує гнучкість у запитах до даних.

Приєднуйтесь до нас, щоб дізнатися більше технології .NET 9, які формують майбутнє розробки програмного забезпечення!
